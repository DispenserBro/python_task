# python_task

- Задание:
```
Задайте список из нескольких чисел,
Напишите программу, которая найдёт сумму элементов списка, стоящих на нечётной позиции
```
- Решение:
```
1. Определяется ф-ия sum_on_odd_pos
2. Внутри ф-ии определяется локальная переменная result
3. При помощи цикла for и ф-ии range к переменной result прибаляются эл-ты на нужных позициях
4. Ф-ия sum_on_odd_pos возвращает переменную result
```
