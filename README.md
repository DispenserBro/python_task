# python_task

- Задание:
```
Задайте список из нескольких чисел,
Напишите программу, которая найдёт сумму элементов списка, стоящих на нечётной позиции
```
- Решение:
```
1. Определяется ф-ия sum_on_odd_pos
2. Внутри ф-ии расчитывается сумма результирующего списка
3. При помощи list comprehension и функции `sum` находится сумма эл-тов
4. Ф-ия sum_on_odd_pos возвращает результат
```
